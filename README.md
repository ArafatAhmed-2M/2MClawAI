## 🌟 Introduction

**2M Claw** is a next-generation Agentic OS designed to bridge the gap between powerful Large Language Models and your local Windows environment. While inspired by the orchestration capabilities of OpenClaw, 2M Claw is built from the ground up to provide a **GUI-first, zero-friction experience**.

Whether you're a developer looking to automate your workflow, or a power user who wants an AI that actually *does* things—like editing code, managing files, or running background tasks—2M Claw is your ultimate autonomous companion.

## 🚀 Key Innovation: The "Black Hole" Dashboard

The center of 2M Claw is its **Black Hole OS Dashboard**. This isn't just a chat interface; it's a command center where you can:
- **Hot-Swap Models**: Switch between OpenAI, Claude, Gemini, and local Ollama models mid-conversation.
- **Visual Skill Management**: Browse the Agentic App Store and install pre-built skills with one click.
- **Automation Scheduler**: Use the built-in Cron Engine to schedule AI tasks (daily digests, market scans, repo audits).
- **Proactive Intelligence**: 2M Claw doesn't just wait for you; it wakes up, "dreams" about your goals, and consolidates memory while you sleep.

## 🛠️ Quick Installation

### Windows (Recommended)
1. Clone the repository: `git clone https://github.com/ArafatAhmed-2M/2MClawAI.git`
2. Enter the folder: `cd 2MClawAI`
3. Run the installer: Right-click `setup/install-windows.ps1` -> **Run with PowerShell**.
4. Launch the Dashboard: Visit `http://localhost:3000` in your browser.

### Verification
Run the system diagnostic tool to ensure your environment is ready:
```bash
npm run doctor
```
