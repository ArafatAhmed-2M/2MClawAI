# Contributing to 2M Claw 🐾

First off, thank you for considering contributing to 2M Claw! It's people like you who make the Agentic AI community amazing.

## 🚀 How Can I Contribute?

### Reporting Bugs
- Use the **Bug Report** template on GitHub.
- Provide a clear description, reproduction steps, and logs from `npm run doctor`.

### Suggesting Enhancements
- Use the **Feature Request** template.
- Describe the "Why" and "How" of the feature.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Ensure your code passes TypeScript compilation: `npm run build`.
3. If you've added a new skill, include an example in the `/skills` folder.
4. Update documentation if your change affects usage.
5. Open the PR with a clear title and description of the problem it solves.

## 💻 Development Workflow

1. **Install**: `npm install`
2. **Build**: `npm run build`
3. **Test**: `npm run dev` (Check the Web UI)

## 🎨 Style Guide
- Use **TypeScript** for all backend logic.
- Follow **CamelCase** for variables and **PascalCase** for classes.
- Keep components modular (don't bloat `server.ts`).
