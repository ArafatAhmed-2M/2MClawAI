# 2M Claw Features

## 1. Web Dashboard GUI
Unlike the original OpenClaw which relied heavily on CLI and YAML, 2M Claw provides a rich, dark-themed HTML/CSS dashboard.

## 2. Multi-LLM Support
Switch instantly between different models from the Web UI:
- OpenAI (GPT-4)
- Anthropic (Claude 3)
- Google (Gemini Pro)
- Groq (Llama 3)
- Ollama (Local Models)

## 3. Skill Browser
Visually browse, install, and configure skills directly from the dashboard instead of managing JavaScript files manually.

## 4. Proactive Agent
Set up scheduled tasks using cron syntax. 2M Claw will execute these tasks automatically and can send you digests or alerts via your connected messaging platforms.

## 5. Persistent Long-Term Memory
2M Claw features an advanced JSON-backed storage engine to maintain state and context across reboots.
