# 🚀 2M Claw: Advanced Agentic Features

2M Claw is designed to be more than just a wrapper; it is an **Autonomous Execution Layer** for Windows. Below are the core features that define the platform.

---

## 🖥️ 1. Black Hole OS (Web Dashboard)
The primary interface for 2M Claw is a high-performance, dark-mode Web UI.
- **Real-time Terminal Logs**: Monitor every thought and action the agent takes.
- **Hot-Connect Bots**: Input your Telegram or Discord tokens and connect instantly without restarting the server.
- **A2A Protocol**: Support for "Agent-to-Agent" communication cards.

## 🤖 2. Multi-Provider Orchestration
2M Claw supports virtually every major LLM provider with intelligent fallback logic.
- **Cloud Providers**: OpenAI (GPT-4o), Anthropic (Claude 3.5), Google (Gemini 1.5 Pro).
- **Speed Kings**: Groq & Together AI for near-instant responses.
- **Local Privacy**: Native support for **Ollama**, allowing you to run 2M Claw entirely offline.
- **Router Support**: Deep integration with **OpenRouter** for access to hundreds of open-source models.

## 🔌 3. Dynamic Skill System
The "Agentic App Store" allows you to extend 2M Claw's capabilities effortlessly.
- **Pre-installed Core**: Filesystem access, Persistent Memory, and Messenger Sync.
- **Extension Logic**: Drop any `.js` skill into the `/skills` folder to give your agent new tools (e.g., SSH, Web Scraping, Database Management).
- **Auto-Loading**: Skills are detected and loaded on boot, making them instantly available to the LLM's system prompt.

## 🕒 4. Automation & Cron Engine
Turn your AI into a proactive employee with the built-in scheduler.
- **Visual Interface**: Create recurring tasks (Daily Digests, Repo Backups, Security Audits) via the GUI.
- **Action Chaining**: Use the LLM to process data and then trigger filesystem or messaging actions automatically.
- **Memory Consolidation**: Scheduled "Dream Cycles" where the agent distills conversation logs into long-term facts.

## 🩺 5. System Doctor & Self-Healing
A dedicated diagnostic layer keeps your Agent OS healthy.
- **Diagnostic Tool**: Run `npm run doctor` to check environment variables, Node.js versions, and permissions.
- **Auto-Updater**: On every boot, 2M Claw checks GitHub for core stability patches and offers to update itself.
