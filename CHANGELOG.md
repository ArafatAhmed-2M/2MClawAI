# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - Initial Release
### Added
- Complete TypeScript backend architecture.
- Express-based Web Dashboard with modern dark-theme HTML/CSS/JS.
- Multi-LLM provider support (OpenAI, Claude, Gemini, Groq, Ollama).
- Visual LLM switcher in GUI.
- Conversation history viewer UI.
- Visual skill browser and installer.
- Scheduled tasks/cron UI.
- Auto-updater utility.
- Long-term memory storage engine.
- Proactive Agent for scheduled digests.
- Native Windows setup wizard (`install-windows.ps1`).
- Unix bash installer (`install-unix.sh`).
- Full project documentation and GitHub templates.
